# ⚡ AQA AutoFill Pro — Installation Guide

## 📁 What's Inside This Folder
```
autofill-extension/
├── manifest.json     ← Extension config
├── popup.html        ← Your profile UI (the popup window)
├── popup.js          ← Saves/loads your data
├── content.js        ← The auto-fill engine
└── icons/            ← Extension icons
```

---

## 🚀 How to Install (Brave / Chrome / Edge)

> ✅ NO hosting needed. Runs entirely in your browser. 100% free & private.

### Step 1 — Open Extensions Page
| Browser | URL to type in address bar |
|---------|---------------------------|
| **Brave** | `brave://extensions` |
| **Chrome** | `chrome://extensions` |
| **Edge** | `edge://extensions` |

### Step 2 — Enable Developer Mode
- Look for **"Developer mode"** toggle in the top-right corner
- Turn it **ON** ✅

### Step 3 — Load the Extension
- Click **"Load unpacked"** button (top-left)
- Navigate to and select the **`autofill-extension`** folder
- Click **"Select Folder"**

### Step 4 — Done! 🎉
- You'll see **"AQA AutoFill Pro"** appear in your extensions list
- Click the puzzle piece 🧩 icon in your browser toolbar
- Pin AQA AutoFill to your toolbar for easy access

## 💡 How to Use

### First Time Setup:
1. Click the **⚡ AQA AutoFill** icon in your toolbar
2. Go to **👤 Personal** tab → Fill in your name, email, phone
3. Go to **💼 Professional** tab → Fill in job title, company, LinkedIn bio
4. Go to **📍 Address** tab → Fill in your address details
5. Click **"💾 Save"** on each tab

### Auto-Filling Forms:
1. Open any job application page (LinkedIn, Indeed, Workday, etc.)
2. Click the **⚡ AQA AutoFill** icon
3. Click **"✦ Fill All Fields Now"**
4. Watch the magic happen! ✨

## 🔒 Privacy & Security
- All your data is stored **locally in your browser only**
- Nothing is sent to any server
- Only YOU have access to it
- Data persists even after browser restarts

---

## ❓ Troubleshooting

**Fields not filling?**
- Make sure you saved your profile first (click 💾 Save)
- Try refreshing the page, then click Fill again
- Some sites load forms dynamically — wait for the page to fully load

**Extension not showing?**
- Click the 🧩 puzzle icon in toolbar → Pin AQA AutoFill

**Want to update your info?**
- Just open the popup, edit any field, click Save
- Changes apply immediately next time you fill
